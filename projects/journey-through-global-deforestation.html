A Journey Through Global Deforestation

To kick off this Udacity SQL Nanodegree project, I was provided data in three tables: forest_area, land_area, and regions. Utilizing my DQL skills, the objective was to address a set of prompts from Udacity to understand the impact of deforestation across countries and regions.

Skills used

  • Left Joins, Inner Joins and Self Joins
  • CTE's
  • Windows Functions
  • Aggregate Functions
  • Creating Views
  • CASE Expressions

Data source

The dataset was provided as part of Udacity's SQL Nanodegree program. It was sourced from the World Bank.

Project reflection

This project marked the first instance where I applied all of my DQL skills simultaneously. Initially, I felt overwhelmed with determining the most suitable DQL command for each prompt. As time progressed, I found my momentum and gained a better understanding of the pros and cons associated with different approaches. I frequently revisited and refined queries, aiming for greater conciseness in my expressions.

Given additional time, I would have delved into exploring my own questions about the data, such as analysing year on year trends on a micro level for each country or region.
